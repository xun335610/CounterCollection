# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for everyone.

## Be respectful

- Be kind and constructive
- Assume good intent, but respect boundaries
- No harassment, threats, or hateful content

## Scope

This Code of Conduct applies within project spaces and in public spaces when an individual is representing the project.

## Enforcement

If you experience or witness unacceptable behavior, open an issue or contact the maintainers.
