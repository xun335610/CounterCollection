# Contributing

Thanks for your interest in contributing!

## Ground rules

- Keep contributions **non-adversarial**: this project is for documentation and calm communication, not for harassment or evasion.
- Do not submit or request real personal data. Use synthetic examples only.

## Good first contributions

- Fix typos or improve US English clarity
- Add or refine **collection-method categories** (descriptive, neutral)
- Improve UI/UX and accessibility
- Add tests where appropriate

## Development

```bash
npm install
npm run dev
```

## Pull requests

- Keep PRs focused and small
- Explain the change and the motivation
- Include screenshots for UI changes
